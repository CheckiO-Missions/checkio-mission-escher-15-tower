これが最後の仕事です! あとはハイパーキューブを取り巻いている小さな力場を取り除くだけです。 そうすればあとは自由に帰還できます。 (無論、予期せぬ困難が無い限りですが)

ハイパーキューブを閉じ込めている筒の周りに多彩な各面を持つ複数の小さな立方体があります。力場を取り除くのにこれらを使うのはほぼ確実と思われます。

あなたの仕事はこれらの立方体で４つの側面が同じ色になるように作成可能な塔の高さの最大値を返すことです。(4つの側面は立方体自身の各面からなる 4色となります)
立方体は１つの大文字で各面の色が表されています。 (A - アズール(紺碧), B - 青, C - シアン(薄水色), G - 緑, O - オレンジ, R - 赤, S - スカーレット(緋色), V - 紫, W - 白, Y - 黄)

面の順番は以下の通りです: 前面, 右側面, 左側面, 背面, 上面, 底面。 立方体の各面の色は重複しません。
あなたは塔を作るのに各立方体を回転させられます。

入力: 各立方体を記述した文字列の配列

出力: 作成可能な塔の高さの最大値

{% endif %}
